InWeCrypTO API文档

简介

RESTful API
版本 v2
返回数据格式,如下:

    {
        "msg": "操作成功",
        "code": 4000,
        "url": "",
        "data":
    }

请求头包含如下信息:

    {
        "Authorization": "", // 用户认证信息
    }

请求url: http://xxx.xxx.xx/v2/
所有分页列表,可以用per_page参数控制分页大小,默认10
